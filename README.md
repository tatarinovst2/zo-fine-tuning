# Application of Zeroth-Order Optimization Methods for Fine-Tuning of Large Language Models

This repository contains code for the thesis
"Application of Zeroth-Order Optimization Methods for Fine-Tuning of Large Language Models"
by Maksim Tatarinov.

It contains these modules:
[dataset_processing](dataset_processing/README.md) - for downloading and preprocessing datasets,
[pipeline](pipeline/README.md) - for fine-tuning and evaluating the model,
[latex](latex) - for the thesis write-up,
[visualization](visualization/README.md) - for visualizing training curves and comparing runs.

Start with the [dataset_processing](dataset_processing/README.md) module to download and preprocess datasets,
then use the [pipeline](pipeline/README.md) module to fine-tune and evaluate the model.
Finally, use the [visualization](visualization/README.md) module to visualize training curves and compare runs.

Don't forget to install the required dependencies from `requirements.txt` before running any code.
